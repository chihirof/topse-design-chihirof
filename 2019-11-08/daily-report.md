### パスについて
- 起動時はroot
- ファイル名、ディレクトリ名は、相対パス、絶対パスを使用できる

### コマンド
- echo <文字列>
- cat <ファイル名>
- ls
- ls -l
    - ファイル名、作成日時
- mv <ファイル名> <ファイル名>
- mv <ファイル名> <ディレクトリ名>
- cp <ファイル名> <ファイル名>
- cp <ファイル名> <ディレクトリ名>
- touch <ファイル名>
- rm <ファイル名>
- mkdir <ディレクトリ名>
- rmdir <ディレクトリ名>
- cd <ディレクトリ名>
- cd -
- history
